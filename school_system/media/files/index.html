CSS Box Model

CSS Box Model is a box that wraps around every Html elements like:-
margins,borders,padding and the actual content

Properties of a CSS box model

Content

This is where the text and images appear

Padding

Clears an area around the content,it is transparent

Border

Border goes around the padding and content

Margin

Margin clears an area outside the border,and it is tranparent.

Calculate the total dimension of a box

400 px(Width)
+40px(left padding+right padding)
+4px(left border+right border)
=444
